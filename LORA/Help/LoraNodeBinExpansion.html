LoraNode Expansion for binary data

Basic LoraNode class methods send and receive char messages (i.e. null terminated string). If you need to send bytes buffer you can use next functions that are binary version of write and receive methods..

 

Write a binary message to a node

writeMessageByte()

Format

bool writeMessageByte(int dest, byte message[],int messlen, int timeout) : send "message" to node "dest" and can wait for air free "timeout" milliseconds maximum; it returns false if wasn't able to send message. (binary version of writeMess())

Check if a binary message is arrived

newMessByteAvailable()

Format

bool newMessByteAvailable(int sender, byte binbuff[],int bufflen,timeout) : check for "timeout" millisecons if message is incoming, but only message from a specific "sender". Binary version of newMessAvailable().

In this case it doesn't use default buffer, but a user supplied buffer instead. Note that decoded message received start at position given by next function.

Get message pointer (byte) just received

getMessageByte()

Format

byte* getMessageByte() : returns pointer to starting byte of message received.

Get num of bytes just received

getNumByteReceived()

Format

int getNumByteReceived() : returns num of bytes actually received.
