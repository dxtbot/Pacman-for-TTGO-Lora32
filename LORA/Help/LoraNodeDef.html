LoraNode Setting

Default values of LoraNode instance can be changed by these functions.

Set/get node id (after instance)
Format

setNodeAdd(unsigned int nodeAdd);

int getNodeAdd();

Set/reset automatic acknowledge
Format

setAutomaticAck(bool yes);

Set/reset automatic acknowledge
Format

setAutomaticAck(bool yes);

Set/get transmission power (use code from 1 to 5)
Format

setPower(byte code);

int getPowerCode();

setAutomaticAck(bool yes);

Set key for cryptography
Format

setKey(int key);

Set/get network id (it defines a network)
Format

setNetId(unsigned int id);

unsigned int getNetId();

unsigned int getMaxId(); range of possible net id following the defined net structure (see below max devices)

Set/get max addressable devices (it defines also the range of possible net id)
Format

setMaxDevices(byte code); code (from 3 to 13) (max devices=2^code) (see LORA page for network structure)

unsigned int getMaxDevices();

Set/get frequency (in MHz)
Format

setFrequency(float nMHz);

float getFrequency();

Set spreading factor
Format

setSpreadingFactor(byte code); code from 6 to 12

Set band width
Format

setBandWidth(byte code); code from 0 to 9

Change receiving buffer length (def. 64)
Format

changeMessageBufferLen(int maxMesslen);

Utility

Print configuration (radio module) (on Serial)
Format

printConfig();

Print network data (adresses)(on Serial)
Format

printAddresses();

Test if air free
Format

bool freeAir();
