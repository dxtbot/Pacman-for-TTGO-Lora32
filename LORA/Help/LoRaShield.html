LoRa shield library

Arduino library for LoRa shield .

This library concerns the LoRa Arduino shield that uses the SX1278 SEMTECH tranceiver or equivalent as RFM98. But you can use also for others LoRa Arduino sistems like remote control or stand alone micro LoRa system (Maduino).

 

Principal class : LoraNode

This class uses LORA class but simplifyes creation and management of a node insert in a crypted network .

 

And other three classes (for detailed uses):

  • LORA ; simplified functions for LoRa modem
  • SX1278 ; basic modem management.(Used by LORA class)
  • REMOTEC ; simplified functions for using SX1278 as OOK transitter with HX2262 protocol

 

Both LORA class and REMOTEC class use basic SX1278 functions. So, you can use SX1278 functions mixed with LORA or REMOTEC functions for better trim.

SX1278 class is referenced by external name SX.

 

Pin used by shield:

  • D13 : SPI SCK
  • D12 : SPI MISO
  • D11 : SPI MOSI
  • D10 (or D8) : SPI CS
  • D5 (or D7) : SX1278 RESET (D9 in new shield and Maduino
  • (D6 or D9 or none) : Inp/Out pulses
  • (D3 or none) : interrupt (D2 in new shield and Maduino
